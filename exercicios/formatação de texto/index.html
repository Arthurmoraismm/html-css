Principais formatações de texto

Negrito/destaque

Nessa frase temos uma frase em negrito usando a tag <b>
porem esse jeito é UM JEITO NÃO SEMANTICO

Tambem podemos deixar uma frase em negrito usando a tag <strong>
E esse é UM JEITO SEMANTICO

Italico / Ênfase

Nessa fraze temos um termo em italico usando a tag
porem esse jeito é UM JEITO NÃO SEMANTICO <i>

Tambem podemos deixar um termo em italico usando a tag <em>
E esse é UM JEITO SEMANTICO

Texto marcado

podemos criar um texto marcado usando a tag <mark>

E no outro paragrafo temos outo texto marcado com a cor original

Texto grand / Texto pequeno

podemos criar um texto grande usando a tag <big> (não semantico)
e podemos criar um texto pequeno usando a tag <small>

Texto Excluido / Texto Inserido

podemos marcar um texto que esta ali mas não deve ser considerado com a tag <del>

podemos marcar um texto inserido com a tag <ins>

TEexto sobrescrito / Texto subescrito

para adicionar um elemento sobrescrito x20+22 basta usar a tag <sup>

E para adicionar um elemento subescrito h2O basta usar a tag <sub>
